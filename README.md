# Odin Recipes

A small, static recipe site.

- Built entirely with **HTML** (no JavaScript, no CSS frameworks).
- Entry point: `index.html`.

## Download / Get the code

### Option 1: Download as a ZIP (GitHub)

1. Open the repository page on GitHub.
2. Click **Code** → **Download ZIP**.
3. Unzip the folder.

### Option 2: Clone with Git

```bash
git clone <YOUR_REPO_URL_HERE>
cd odin-recipes
```

## Run locally

Because this is plain HTML, you can run it without installing anything:

- Open `index.html` in your browser.

Optional (nicer): use a local server (e.g., VS Code “Live Server”) so relative links behave consistently.
